# final
+ final修饰方法时，表示此方法不能被子类重写，但是该方法人可以随当前类被继承

# round方法
+ ceil向上取整，比如`11.3->12`,`-11.3->-11`
+ floor向下取整
+ round加0.5后向下取整，比如`-11.3->-11`,`-11.6->-12`

# String类常用方法
+ 获取功能（长度、字符、子串、字符首次出现索引）
+ 判断功能（判同、是否包含子串、空串、以...开头结尾）
+ 转换功能（byte数组、字符数组、在转回来、大小写转换、拼接）
+ 其他功能（字符、子串替换、两端去空格等）

# new String("a") + new String("b") 会创建几个对象？
+ 1、new StringBuilder() 体现在“+”
+ 2、new String(a) 
+ 3、常量池中的"a"
+ 4、new String(b)
+ 5、常量池中的“b”
+ 6、注意：StringBuilder最后要转回String，调用了toString方法，toString方法只会在堆中创建对象“ab”,不会在常量池中生成“ab”
